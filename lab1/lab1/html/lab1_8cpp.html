Author

Description:
This function receives an array and will process the array placing the elements in order using the bubble sort method. It moves through the array comparing adjacent items and swapping the elements if they are not in order. It will make n number of passes. If it makes a complete pass without swapping two elements, the array is in order and no further processing is required.

Description:
Will shift elements up in the sorted half of the array as long as the item is larger than the item being inserted. Once it encounters an item that is <= to the number being inserted, it will stop shifting and insert the item into the vacated spot. this will happen n number of times in order to insert n items into the array.

int main ( int  argc,
char **  argv 
)
Author
Fred Flintstone
Description:
This is the starting point to the program. It will retreive a number form the rand function and place it into three arrays. Each array will then be sorted using one of three sort techniques, bubble, insertion, and selection. Upon returning from each sort function the sorted array will then be verified using the isSorted function. If the array is not in order, a message will be outputted and an error number will be returned.
Parameters
[in]argc- the number of arguments from the command prompt.
[in]argv- a 2d array of characters containing the arguments.
Returns
1 bubble sort failed
2 insertion sort failed
3 selection sort failed
