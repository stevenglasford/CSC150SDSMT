Program Information

This program asks the user for 3 pairs of octave and pitch class values. The computer then determines if the octaves are between 0 and 11, and if the pairs can be heard when passed through a frequency function. If and only if the octaves are between 0 and 11 as well as the sound being able to be heard, will the computer produce the tones.
